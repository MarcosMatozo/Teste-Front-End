# bexs-front-end-exam

### Configuração inicial do projeto - para dev
```
npm install
```

### Compilador e hot-reloads - para dev e visualização da aplicação
```
npm run serve
```

### Build para produção
```
npm run build
```

### Lints - Erros
```
npm run lint
```
